# CleanByte

CleanByte is a sleek, responsive web application designed to be the main landing page for CleanByte laptop cleaning service.
This project focuses on a Responsive Web Design approach and a minimal, user-friendly UX using the Standard Web Stack (HTML, CSS, JS).

## 💼 The Business
CleanByte is a professional laptop maintenance and cleaning service founded by a team of students at **UTM SPACE**. This project serves as both a live business venture and a technical implementation for our Entrepreneurship & Business assignment.

## 🌐 The Website
### 🚀 UI Features

* **Minimalistic Design:** The website's UI elements are designed to be simple and clean, yet expressive to allow users to effectively interact with them.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
* **Single-Page Architecture:** Everything the user needs is contained in one continuous flow, eliminating load times between sections.

### 📂 Project Structure

* `/css`: Contains modular stylesheets for different sections.
* `/js`: Logic for smooth scrolling, dropdown interactions, and utility scripts.
* `/src`: Assets and additional page resources.
* `index.html`: The structural backbone of this whole page.
* `terms.html`: a dedicated page just for our business's terms and conditions.

1.  **Clone the repository:**
